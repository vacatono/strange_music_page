ポセイドンのイベントです。この日はバッティングしまくり（小川美潮さんとか、4-Dとか、NATSUMEN+MONG HANGとか）でしたけど、とりあえずこちらへ。

### Cherno

このバンドだけは初めて見ます。高円寺とかでやってるみたいです。
んーと、音的には嫌いじゃないけど。やっぱし生ドラム入れて欲しいな、出来ればベースも。

- 菅原真 (sax)
- 岸本淳一 (guitars)

( http://www.asahi-net.or.jp/~kv6j-ksmt/ )

### Djamra

去年の夏に大阪で見て以来。（その時は色々混じった混成一個旅団みたいな編成だったけど）
バスドラの響きが気持ちイイなーとか思って踊ってたら、いつのまにか横に居たこないだ見た高円寺百景の鍵盤弾きにツッコまれた。
途中で演奏した「あるはあは」って曲がシミジミと良かったです。ネタとテクニックで押し切るだけのバンドじゃなくって良かった（って何か偉そうだな）
で、会場で売ってた3rdのCDを買ってきた。さっき開けたらライナー内側の写真の馬鹿さ加減に大笑いしした。

- 中来田正晴 (bass)
- 北村信二 (sax)
- 石川晃 (guitars)
- 福田岳彦 (keyboards)
- 榎本明宏 (drums)

( http://park8.wakwak.com/~djamra/ )

### FLAT122

えーとゲスト無しは久々かな？この日初めて聴いたのは「記憶」って新曲。このバンドとしては比較的シンプルな感じの曲でした。綺麗なメロディーが伝わりやすくてイイかも。
そいえば今回は見ててハラハラするよーな所が無かったな。なんだか無茶な曲なんだけど無理矢理演奏してるような感じじゃなくなったみたい。安定してきたって事かな？
で、なんだか盛り上がってアンコール。「書くでしょ」とか言われたら書くしかないよなー、アンコールとはいえ長い曲しかないわけで・・・急遽「夏」を演奏。

- 川崎隆男 (piano)
- 平田聡 (guitars)
- 田辺清貴 (drums)

( http://music.geocities.jp/thewaves2005/ )

### HARPY

あの奇矯でゆるーい感じも好きだけど、ちょびっとプログレ臭のする初期の曲がやっぱり好きだなぁ。とか最後に演奏した「ピ」って曲を聴きながら思った。

- イトケン (drums)
- きょうこ (vocals)
- オノテツ (keyboards)
- ズッキー (guitars)

( http://www.itoken-web.com/Harpy/ )
