連休明けの微妙な客入りな感じでしたが、久々に渋谷LUSHへ。着いたら最初のバンドが終る辺り（ノイジーなNEU!みたいな音だった）でした。

### Circuit Unconnection

simの大島さんとFLYの山田民族さんと植村さん・・・ベースの福田亮さんは前に何か他のバンドで見たような・・・
CD-R買ってて気に入ってたのですけど、ナマで見るのはこのバンドも初めて。いつもの如くなんて説明したらイイのか良く分からんけど、物凄く極端なコンポジションをギリギリまで再現するよーな演奏。<a href="http://www.geocities.jp/oosim2000/">音源</a>聴いてもらった方が早いと思うのですが、ちょっと独特なグニャっとしたノリが気持ち良いです。

　大島輝之 (guitars)
　山田民族 (guitars)
　福田亮 (bass)
　植村昌弘 (drums)

　
( http://air.ap.teacup.com/en-junak/ )

### AZMARI

手数の多いドラムと軽くジャズっぽい鍵盤と若干70年代っぽい雰囲気のギターの3人編成のジャズロック。ベースレス編成なので鍵盤の左手でベースパートをカバーしたりギターの人がオクターバー使ってベースパートやったりしてます。特に鍵盤の方のベースラインがちょっと独特で面白かったです。
特に最後の曲とかプログレ色強くて、ギター主導のインストでちょっとFINCHとかAIN SOPHみたいな雰囲気も、結構気に入って手売りのCD買ってきちゃいました。
（ちなみにFINCHってのはオランダのプログレバンド・・・マイナーかな）

　渋谷文平 (guitars)
　森田宏史 (keyboard)
　大路貴久 (drums,glockenspiel)

( http://www.geocities.jp/azzzmari/ )
( http://www.myspace.com/azzzmari )

### te_ri

こちらは何度か見てますが、村上さんと片山さんのデュオ。
アンプ直結のギターと乾いた音のドラムで、またコレも妙な所にアクセントの入る変なリズムの音楽。緊張感漂う音なんですけど、一瞬フワっとメロディアスになる瞬間が面白かったり。
いやホントに毎回結構面白いと思うんですけど。

　村上巨樹 (guitars)
　片山貴志 (drums)

( http://www.myspace.com/teritokyo )
