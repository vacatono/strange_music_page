[ tanta ]

結構面白い混成楽団なのでカッコの中にバンド名を書いておきます。
思ってたよりも拡散しない感じのセッション風の音でした。ベースの人だけ初めて見たのですけど、しっかり自己主張しつつも安定していて良かったです。
あとHumbert Humbertの佐藤さんがこんな即興っぽいことやるのが不思議な感じだったのですが結構ハマってました。
割と良かったのですけど、盛り上がってくると大体誰かが抑える方向に行っちゃうのがちょっと残念・・・もっとガーっと行ってしまっても良いのにとか思いました。

- 佐藤良成[Humbert Humbert] (fiddle)
- 根本潤[HUNUNHUM,Z] (sax)
- 藤井友信[music from the mars] (guitars)
- 中俣和宏[downy,Karen] (bass)
- 山本達久[zggzag!!!] (drums)

[ 山川冬樹+HIKO ]

ハードコアのドラムに絶叫ガピピピピなボイスパフォーマンス。ドラムの方はGAUZEってバンドの方みたい。

- 山川冬樹 (voice)
- HIKO (drums)

[ HUNUNHUM ]

ちょっと久々のHUNUNHUM。相変わらずのいつもの曲ですけど、今回結構細かいところが変わってたかも。
CDはまだみたいですけど、今回はDVDを作って持ってきてました。とりあえず当然買った。
前は「ハードコアなナショナルヘルス」だと思ってましたけど「どっちかとゆーとPicchio Dal Pozzo」って気がしてきた。
とにかく今一番面白いと思ってるバンドなので、MYSPACE ( http://myspace.com/hununhum/ ) 辺り見てみてください。

- 中川達也 (guitars,voice)
- 木村友紀彦 (drums,percussions)
- 岩清水雄 (bass)
- 根本歩 (percussions,handsonic)
- 根本潤 (sax,recorder,voice)
- 多田佳代 (keyboards,voice)

[ ドラびでお+灰野敬二 ]

・・・なんちゅー組み合わせよ？
「ドラびでおplays灰野敬二vs灰野敬二」って文字通りそんな感じの音と映像でした。
灰野さんの映像を一楽さんのドラムでコントロール、どらビデオ分だけでも十分うるさいのに、そこに相変わらず灰野さんのギターが（ボイスもか）ズギャギャギャと。
まぁ面白かったけどねー。

- 一楽儀光 (drums)
- 灰野敬二 (guitars)
