NAUTILUS RECORDS（辻伸夫さん主催のレーベル）のイベント。PA卓に新月の津田治彦さんが！！！

### 神谷きよみ

この日はピアノ弾き語りだけじゃなくて、チェロ×２＋バイオリンのアンサンブルでの曲。
可愛いチェンバーポップみたいで良かったです。んーと、またインストやって欲しいなぁ。

- 神谷きよみ (piano,cello,vocals)
- 斎藤孝太郎 (cello)
- 武藤祐生 (violin)

( http://kkiyomi.com/ )

### マコフェイ

鍵盤＋ボーカル＋テルミンでの演奏。ボーカルのスタイルはあんまり好きな感じゃないのですけど、曲が凄く良い。あと鍵盤の方が地味なんだけど凄く凝ってて面白かった。
後半からMa\*Toさんがタブラで参加。KILLING TIME以外でMa\*Toさんのタブラを聴くのってなかなか無いので貴重だったかも。

- フェイターン (vocals,thermine)
- 白川眞琴 (keyboards)
- Ma\*To (tabla)

( http://macofei.konton-records.com/ )

### 生方ノリタカ

またテルミン奏者さんです。最初は「なんだか変なオッサンがトリなんだなー」とか思ったのですけど、
微分音階ポップス＋テルミンって感じの音で、奇妙にポップでちょっとトホホな感じの物凄く変な音楽。
「SYZYGYSみたいだなー」とか思ったのですけど、あんまり知ってる人いないかな？ ( http://www.syzygys.jp/index.html )
特に「泥の島 ( Mud Island )」って曲が「チューニングの狂ったピロレーター」みたいな感じでサイコーでした。CD買っちゃった。
（ http://www.nextmusic.net/index.php?command=profmusic&profid=20070919112733 ←泥の島サンプル )

アンコールは、全員参加でテルミンがニャーニャー言う曲と、何故かPink Floydの狂気から・・・何だっけなアレ？「The Great Gig In The Sky」だっけ？またもー、何か微妙なセレクションで。素晴らしい演奏を。

- 生方ノリタカ (thermine,computer)

( http://ubuman.sakura.ne.jp/ )
